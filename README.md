git clone https://**github.com/riku2202/ContentProduction2.git**

◇ブランチの操作

・ブランチの作成　　　　
git branch ブランチ名

・ブランチの切り替え　　
git checkout ブランチ名

・ブランチの削除　　　　
git branch -d ブランチ名

◇プロジェクトを最新の状態にする

git fetch origin

git merge origin/ブランチ名

◇プロジェクトの更新をリモートにアップする

git add .

git commit -m "作業内容"

git push origin ブランチ名

※一度だけ[ git push -u origin ブランチ名 ]を実行すると
いつも通りの形[ git push ]で実行することができます
